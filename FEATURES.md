# 🌟 CareerLens AI Features

CareerLens AI is built with a focus on both high-end design and practical AI utility. Below are the key features and design elements.

## 🧠 AI-Powered Career Guidance
- **Resume Analysis**: Uses Claude 3.5 Sonnet to parse and analyze PDF resumes.
- **Skill Gap Identification**: Compares user skills against industry standards for their target role.
- **Strategic Recommendations**: Provides actionable steps to improve employability.

## 🎨 Premium Student Portfolio Aesthetic
The application follows a "Workspace/Portfolio" design language rather than a standard SaaS look:
- **10 Ambient Elements**: Dynamic background elements (floating particles, glassmorphism cards, ambient glows) that create a rich depth.
- **Micro-interactions**: Hover effects, smooth transitions, and tactile feedback on interactive elements.
- **Modern Typography**: Uses Inter and Geist fonts for a clean, academic-yet-tech-forward look.

## 🎬 Advanced Animation System
- **Scroll Reveal**: Elements fade and slide into view as the user scrolls, creating a guided narrative.
- **Staggered Delays**: Grid items and feature lists appear with a natural, staggered timing.
- **Sticky Progress**: A persistent scroll progress bar and dynamic navbar that adapts to scroll position.

## 📱 Core Pages
1.  **Landing Page**: High-impact hero section with value propositions.
2.  **Features**: Interactive showcase of the platform's capabilities.
3.  **How It Works**: A step-by-step visual guide for new users.
4.  **Stories**: Real-world examples and testimonials.
5.  **Pricing**: Transparent 3-tier pricing model.
6.  **Dashboard**: The central hub for file uploading and analysis results.
